# Laptop Setup

Két sor a Terminálba — mindent telepít és beállít.

```bash
curl -fsSL https://raw.githubusercontent.com/kriso-git/setup/main/setup.sh -o ~/setup.sh && chmod +x ~/setup.sh
bash ~/setup.sh
```

**Mit csinál:**
- Homebrew, Node.js, Git, Vercel CLI (csak ha még nincs meg)
- Klónozza: donna-pizza, alexoldal, f3xykee-terminal
- npm install mindenhol
- f3xykee `.env.local` letöltése Vercel-ről automatikusan
